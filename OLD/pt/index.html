Biografia

Magno Tairone é doutorando em Estatística no Departamento de Estatística da Universidade de São Paulo. Suas áreas de interesse em pesquisa compreendem ciência dos dados, análise de dados em altas dimensões, inferência Bayesiana e estatística espacial. Atualmente é membro do grupo Stats4Good na Universidade Federal de Minas Gerais, um grupo de alunos entusiastas que analisam os dados sobre gastos do governo brasileiro no âmbito federal. Projetos

Veja alguns projetos em que tenho trabalhado.

Visualizando as desigualdades existentes no Brasil atrvés dos resultados do ENEM

Este é um dos projetos que estou atualmente trabalhando. O objetico é analisar e verificar se as desigualdades sociais , que são conhecidas por bastante evidentes no Brasil, se refletem na performance dos candidatos que prestam o Exame Nacional do Ensino Médio (ENEM).

Geojags

Um módulo para o software JAGS, que contém diversas funções de covariância para análise de dados de referênia pontual e uma função que cria a matriz de precisão para o modelo CAR próprio, para a anális de dados de área. O desenvolvimento deste módulo fez parte da minha dissertação de mestrado.

Veraz

Veraz é um projeto do grupo Stats4Good, que é um time de estudantes da pós-graduação em estatística que analisa dados de gastos do governo federal do Brasil. Este projeto é desafiador, pois envolve a manipulação de uma quantidade muito grande de dados. Nós mantemos um blog onde postamos nossas análises.
